# Генератор индивидуальных заданий по машинному представлению вещественных чисел

## Описание
Программа генерирует индивидуальные задания для студентов по теме 
"Машинное представление вещественных чисел". 

## Функциональность
- Чтение параметров из config.txt
- Генерация n вариантов по k заданий
- Поддержка 32, 64, 128 битных вещественных чисел
- Перевод в машинное представление и обратно
- Вычисление точности перевода
- Генерация markdown файлов в каталоги Задания/ и Проверка/
